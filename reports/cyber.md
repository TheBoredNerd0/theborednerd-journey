📚 Learning topic — CTF & PRACTICAL SKILLS:
Today is Day 29 = CTF week. Time to go hands-on.

**What is a CTF?** Capture The Flag competitions simulate real hacking
scenarios. You solve challenges (web, crypto, forensics, binary exploitation,
networking) to "capture" a hidden text string. They're the closest thing to
the real job that won't get you arrested.

**Why it matters for jobs:** CTFs are the #1 way recruiters verify you can
actually hack — far more than any cert alone. Hiring managers at
CrowdStrike, Mandiant, and start SOCs routinely check HackTheBox profiles.

**Start here — pick your level:**
- Absolute beginner: PicoCTF (picoctf.org) — browser-based, built for HS/college
- Beginner → Intermediate: TryHackMe "CTF Playground" & "CTF 101" rooms
- Intermediate → Advanced: HackTheBox (free retired machines, VIP for active)
- Linux fundamentals via CTF: OverTheWire Bandit (overthewire.org/wargames/bandit)

**Practice today (15-30 min):**
Solve 3 PicoCTF "General" challenges — starting with the "Python Wrangler"
or "Wave a signal" warmup. No setup needed, browser-based.
Link: https://play.picoctf.org/practice

**Build the habit:**
Join a CTF team on Discord (Unofficial CTF, ctfd.io community). Many CTFs
run weekends — team events look great on resumes.

💡 Career tip:
Create a GitHub repo called "ctf-writeups". Every challenge you solve,
write a 5-minute walkthrough of HOW you solved it. Frame it like a
blog post. This becomes a portfolio AND a learning tool. Recruiters eat
this up — it proves you can think through a problem, not just run tools.

🛠️ Tool trending this week:
**CyberChef** (GCHQ) — the "Cyber Swiss Army Knife." Encode/decode, decrypt,
