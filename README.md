# 🤓 TheBoredNerd — AI Journey

> Building an AI-powered life from scratch. Documenting every step publicly.

**Started:** 2026-03-27 | **Location:** Singapore | **Platform:** [OpenClaw](https://openclaw.ai)

---

## What is this?

I'm using **[OpenClaw](https://openclaw.ai)** to build a personal AI agent system that runs my life in the background — 24/7, on a Mac mini at home. This repo is the live brain: agents write here, code lives here, and the daily diary is updated automatically at midnight every night.

No hype. Just building.

---

## The Goal

I'm not doing this to look smart. I'm doing it because I want to:

- **Make money** using AI tools legally (affiliate, content, micro-SaaS — figuring it out as I go)
- **Get into cybersecurity** — career goal, learning daily via Cyber_agent
- **Build content channels** on YouTube, TikTok, Instagram
- **Get better at development** — this repo IS the project
- **Document everything** — so if it works, others can copy it; if it fails, they can learn from that too
